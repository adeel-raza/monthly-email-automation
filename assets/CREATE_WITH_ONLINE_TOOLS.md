# Create Banners Using Online Tools

## Quick Method: Canva (Recommended)

1. **Go to Canva**: https://www.canva.com/
2. **Create Custom Size**:
   - Banner: 772x250px
   - High-res Banner: 1544x500px  
   - Icon: 256x256px

3. **Design Elements**:
   - Background: White (#ffffff)
   - Main Text: "Simple Email Scheduler" (Large, bold, #2271b1)
   - Subtitle: "Automate Recurring Emails & Newsletters" (Smaller, #00a0d2)
   - Add email icon (ğŸ“§ or search "email icon")

4. **Export**: Download as PNG

5. **Save files as**:
   - `banner-772x250.png`
   - `banner-1544x500.png`
   - `icon-256x256.png`
